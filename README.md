# anysphere-nvim

A dark Neovim colorscheme based on the Cursor editor theme.

## Variants

- `anysphere` - vibrant colors
- `anysphere-pale` - softer, muted pastels

## Installation

**lazy.nvim**
```lua
{ "asurve/anysphere.nvim" }
```

**packer**
```lua
use "asurve/anysphere.nvim"
```

## Usage

```lua
vim.cmd.colorscheme("anysphere")
-- or
vim.cmd.colorscheme("anysphere-pale")
```
